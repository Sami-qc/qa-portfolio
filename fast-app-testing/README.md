# Fast App Testing Project

## Overview
This project documents exploratory testing conducted on the Fast mobile application.

The focus is on identifying real-world issues in the order flow, especially post-order actions.

---

## Scope
- Order placement
- Current orders
- Order details
- Order tracking
- Cancellation behavior

---

## Key Finding
🚨 Critical Issue: Users cannot cancel an active order after placing it.

---

## Project Structure

- test-cases → contains valid & invalid scenarios
- bug-reports → contains detailed bug reports
- evidence → screenshots & GIFs

---

## Objective
 - identifying bugs
- writing test scenarios
- documenting issues professionally
