# Test Plan (Sample) - SauceDemo


## Objective
Verify the main user flows of the web app and ensure critical functionality works correctly.

## In Scope
- Login
- Product listing
- Add to cart
- Cart page
- Checkout (basic flow)

## Out of Scope
- Performance testing
- Security testing
- Cross-browser matrix (for this sample)

## Test Types
- Smoke
- Functional
- Regression (basic)

## Test Environment
- OS: Windows 10/11
- Browser: Chrome (latest)
- URL: https://www.saucedemo.com
- Test Data: Standard user accounts

## Entry Criteria
- Application is reachable
- Build is stable (no major outages)

## Exit Criteria
- Smoke tests pass
- No open Critical/High bugs for core flows

## Risks
- Demo site instability
- Test data changes
