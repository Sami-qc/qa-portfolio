# Test Scenarios — IRAVIN Website

## Feature: Homepage

- Verify homepage loads successfully
- Verify banners are displayed correctly
- Verify clicking banner redirects to correct page

---

## Feature: Navigation Menu

- Verify menu opens and closes correctly
- Verify navigation links redirect to correct categories
- Verify categories display correct products

---

## Feature: Search

- Verify search returns relevant results
- Verify search with invalid keyword shows no results message
- Verify search suggestions appear while typing

---

## Feature: Product Listing

- Verify products are displayed correctly
- Verify discount labels are accurate
- Verify clicking product opens product details page

---

## Feature: Product Details

- Verify product information is displayed (price, name, image)
- Verify selecting size/color works
- Verify Add to Cart button works

---

## Feature: Cart

- Verify product is added to cart successfully
- Verify quantity can be updated
- Verify product can be removed

---

## Feature: Checkout Entry

- Verify user can proceed to checkout
- Verify error shown if cart is empty

---

## Feature: Language & Currency

- Verify switching language works correctly
- Verify currency updates properly

---

## Feature: Responsive

- Verify website works on mobile view
- Verify layout adjusts correctly on different screen sizes
