# IRAVIN Web Testing Project

## Overview
This project presents a structured manual testing assessment of the IRAVIN e-commerce website.

## Website
https://shop.iravin.com/

## Scope
- Homepage and banners
- Navigation menu
- Search
- Product listing pages
- Product detail pages
- Cart and checkout entry flow
- Language and currency behavior
- Responsive behavior

## Objective
To identify functional, usability, and content consistency issues through manual testing.

## Project Structure
- test-plan
- test-scenarios
- test-cases
- bug-reports
- evidence

## Note
This is an independent QA portfolio project created for educational and professional showcase purposes only.
