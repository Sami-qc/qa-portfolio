# IRAVIN Web Testing Project

## Project Overview
Manual testing project for the IRAVIN e-commerce website focused on checkout flow, shipping logic, and pricing validation.

## Website
https://shop.iravin.com/

## Scope
- Cart functionality
- Checkout process
- Shipping rules
- Promotional banner validation
- UI/UX checks
- Business logic validation

## Key Findings
- Free shipping was not applied at checkout for orders above 1500 EGP
- Promotional banner text showed localization inconsistency
- Banner messaging did not match actual checkout behavior

## Test Coverage
This project includes:
- Positive test cases
- Negative and edge case scenarios
- UI/UX validation
- Business logic validation

## Tools Used
- Google Sheets / Excel
- Jira
- Jam.dev

## Files
- `E-commerce Checkout Test Cases - IRAVIN.xlsx`

## Jira Bugs
Tracked bugs include:
- Free shipping not applied at checkout for orders above 1500 EGP
- Promotional banner text does not fully update on language change

## Outcome
This testing effort identified critical issues affecting user trust, checkout accuracy, and promotional consistency.
